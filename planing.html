  • Sección de texto sobre el tema

Que es y para qué sirve el contenido HTML

El contenido o estructura HTML, es un conjunto de etiquetas que se codifican en HTML, dichas etiquetas funcionan para dar una estructura y dividir secciones de nuestra página web de tal manera que podamos tener una organización para cada uno de nuestros contenidos, para explicar esto de una mejor manera podemos imaginar que queremos crear una página web y en la parte central deseamos que vayan un contenido y una barra lateral entonces para esto podemos usar las etiquetas section y aside de las cuales se hablara en este OVA junto con otras etiquetas más que se pueden ajustar a nuestros requerimientos.

Section

Es una sección que hace parte de un contenido que por lo general se ubica centralmente y dentro de esta etiqueta se encuentran elementos que se relacionan entre sí.

Aside

Esta es un contenido que es secundario y por lo general se utiliza para mostrar algunas barras laterales con anuncios o texto complementario.

Footer

Se refiere al pie de página es decir que se puede utilizar como pie de página de toda la página web para mostrar alguna información de derechos de autor o mostrar los medios de comunicación de la página web como por ejemplo correos electrónicos, teléfonos, redes sociales entre otros. Cabe aclarar que este footer también se puede utilizar como pie de página de alguna de las otras secciones que conforman el contenido HTML.

Article

Es una unidad independiente que puede ser utilizada en varias partes de la estructura general, dicha etiqueta puede estar contenida dentro de otra article, por lo general este elemento se utiliza para mostrar alguna información como un post, entrada de un foro o alguna nota.
