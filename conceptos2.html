Article



De acuerdo a la página web aulaClic(2015), article es una unidad independiente que puede ser utilizada en varias partes de la estructura general, dicha etiqueta puede estar contenida dentro de otra article, por lo general este elemento se utiliza para mostrar alguna información como un post, entrada de un foro o alguna nota.
Barzana. (18 de agosto de 2019). Footer. [Imagen]. Recuperado de: https://www.um.es/docencia/barzana/DAWEB/2017-18/daweb-tema-9-estructura-cajas-web.html

Footer



Con relación a la página web desarrolladoresweb(2018) y la página web aulaClic(2015), footer se refiere al pie de página es decir que se puede utilizar como pie de página de toda la página web para mostrar alguna información de derechos de autor o mostrar los medios de comunicación de la página web como por ejemplo correos electrónicos, teléfonos, redes sociales entre otros. Cabe aclarar que este footer también se puede utilizar como pie de página de alguna de las otras secciones que conforman el contenido HTML.
