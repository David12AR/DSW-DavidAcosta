Que es y para qué sirve el contenido HTML


El contenido o estructura HTML, es un conjunto de etiquetas que se codifican en HTML, dichas etiquetas funcionan para dar una estructura y dividir secciones de nuestra página web de tal manera que podamos tener una organización para cada uno de nuestros contenidos, para explicar esto de una mejor manera podemos imaginar que queremos crear una página web y en la parte central deseamos que vayan un contenido y una barra lateral entonces para esto podemos usar las etiquetas section y aside de las cuales se hablara en este OVA junto con otras etiquetas más que se pueden ajustar a nuestros requerimientos.
