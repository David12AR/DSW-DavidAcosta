Que es y para qué sirve el contenido HTML


De acuerdo con Desarrolladores Web (2018),el contenido o estructura HTML es un conjunto de etiquetas que se codifican en HTML, dichas etiquetas funcionan para dar una estructura y dividir secciones de nuestra página web de tal manera que podamos tener una organización para cada uno de nuestros contenidos, para explicar esto de una mejor manera podemos imaginar que queremos crear una página web y en la parte central deseamos que vaya un contenido y una barra lateral entonces para esto podemos usar las etiquetas section y aside de las cuales se hablara en este OVI junto con otras etiquetas más que se pueden ajustar a nuestros requerimientos.
Barzana. (18 de agosto de 2019). Section. [Imagen]. Recuperado de: https://www.um.es/docencia/barzana/DAWEB/2017-18/daweb-tema-9-estructura-cajas-web.html

Section



Teniendo en cuenta la pagina web Desarrolladores web(2018) y aulaClic(2015),Section es una sección que hace parte de un contenido que por lo general se ubica centralmente y dentro de esta etiqueta se encuentran elementos que se relacionan entre sí.
Barzana. (18 de agosto de 2019). Aside. [Imagen]. Recuperado de: https://www.um.es/docencia/barzana/DAWEB/2017-18/daweb-tema-9-estructura-cajas-web.html

Aside



Esta es un contenido que es secundario y de acuerdo a Desarrolladores Web(2018), por lo general se utiliza para mostrar algunas barras laterales con anuncios, banners o texto complementario y adicional a esto ayuda a los motores de busqueda para que se muestre mejor el contenido.
